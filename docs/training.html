🏦 儲互社分析系統

使用者訓練手冊 — 操作 × 解讀 × 預警 × 行動

🎯 目標使用者：社職員、區會、管理員 📱 手機 / 電腦 皆可使用

🔗 立即前往系統 → cu-analysis-v1.streamlit.app

1. 系統簡介

這是一套專為台灣儲蓄互助社打造的風險管理與財務分析平台。 您只需要上傳 Excel 或 CSV，系統就會自動算出每個社的風險狀態、財務比率，並用紅綠燈顏色告訴您哪個社需要關注。

💡 一句話理解這套系統 「把密密麻麻的報表數字，變成一看就懂的紅綠燈與圖表，讓您知道哪個社有問題、問題在哪裡、該怎麼辦。」

1.1 登入方式

1
打開連結 打開管理員給您的網址（或上方連結），會看到登入畫面。
2
輸入密碼
  • 管理員：輸入管理員密碼，可上傳資料、看全台數據
  • 個社 / 區會：輸入專屬密碼（由管理員設定），系統會自動帶入您的資料範圍
3
登入成功 左側會顯示您的身份（🔑 管理員模式 或 👁️ 訪客：ＯＯ社）
⚠️ 密碼錯誤 連續輸入錯誤 5 次會被鎖定，請聯絡管理員。

1.2 三種身份與權限

身份看到什麼可以上傳資料嗎？
管理員 全台所有社的數據 ✅ 可上傳 Excel、CSV，可產生分享連結
區會 所屬區域內所有社的數據 ❌ 不可上傳
個社 只有自己的數據 ❌ 不可上傳
✅ 登入後的差別 — 管理員可看到「全台平均」
— 區會看到「區域平均」
— 個社看到自己的數據，旁邊會顯示「區域平均」作為對照

2. 基本操作

2.1 上傳資料（管理員專用）

管理員登入後，左側側邊欄會有「資料匯入」區塊：

1
準備檔案
Excel 檔：包含三個工作表 — 「社務及資金運用情形」「放款及逾期放款」「區域分類表」
CSV 檔：會計科目明細（科目代號 + 金額）
2
點選上傳
按「Browse files」選取檔案，系統會自動解析。
成功會顯示 ✅ Excel/CSV 解析成功
3
開始使用
上傳成功後，上方會出現「📊 社務診斷」「⚖️ 財報明細」兩個功能，按一下就能切換。
💡 資料更新頻率：建議每月上傳最新資料，系統會自動計算同比（YoY）變化。

2.2 分享連結（管理員專用）

上傳完資料後，側邊欄下方有「生成分享連結」按鈕：

  1. 按 🚀 生成分享連結
  2. 系統會把檔案上傳到雲端，產生一個網址
  3. 複製這串網址，發送給個社或區會
  4. 對方打開連結就會自動載入資料，輸入密碼即可使用
⚠️ 注意 連結中有包含檔案代碼，只限收到連結的人能存取。

2.3 系統導覽

登入後，左側側邊欄有「系統導覽」：

  • 📊 社務診斷 — 風險診斷主畫面（經營總覽、風險矩陣、趨勢）
  • ⚖️ 財報明細 — 財務戰情室（資產負債表、損益表、深度分析）

沒看到某個功能？表示您沒有上傳對應的檔案（Excel → 社務診斷、CSV → 財報明細）。

3. 📊 社務診斷（3 分鐘看懂全貌）

這是系統的核心功能。一進來會看到五個分頁標籤， 我們依序介紹最常用的四個。

3.1 經營總覽 — 第一眼就看這四張卡片

📌 指標 1
社員總數
所有社的社員人數加總。後面括號是增減百分比。
🔺 紅色 = 社員成長（好）
🔻 綠色 = 社員流失
📌 指標 2
股金總額
社員存入的股金總額。
🔺 紅色 = 股金增加（好）
🔻 綠色 = 股金減少
📌 指標 3
開支比
= 總支出 ÷ 總收入。越低越好，超過 100% 表示入不敷出。
🔺 綠色 = 開支比惡化
🔻 紅色 = 開支比改善（好）
📌 指標 4
逾放比
= 逾期放款 ÷ 總放款。越低越好，超過 10% 是高風險。
🔺 綠色 = 逾放比惡化
🔻 紅色 = 逾放比改善（好）
📖 怎麼解讀這四張卡片？
個社模式：顯示「本社」的數字 + 去年比對
區會/管理員：顯示「區域平均/全台平均」，可快速掌握整體趨勢
✅ 四張都往好的方向走（社員增、股金增、開支比降、逾放比降）= 經營穩健

🟢 狀態雷達監控（四色卡片）

在四張 KPI 卡片下方，有四個狀態卡片，這是系統最核心的預警功能：

🚨 特別關懷
5 項檢測中觸發任 2 項以上 → 最需要優先處理
⚠️ 流動性緊繃
貸放比過高（>90%）且股金在流失
💤 資金閒置
貸放比過低（<30%）且逾放比安全 → 錢沒放出去
✅ 穩健模範
社員/股金都成長，貸放比適中（40–80%），逾放比低

點卡片會看到該狀態的所有社別名稱。

3.2 風險矩陣 — 一眼看穿每間社的位置

這是一個泡泡散佈圖，X 軸是「貸放比」、Y 軸是「逾放比」， 每一個泡泡就是一間儲互社。

  • 泡泡大小 = 社員人數多寡
  • 顏色 = 風險狀態（紅/橙/藍/綠/灰）
  • 四象限背景色塊：
    • 🔴 右上角 = 高貸放比 + 高逾放比 → 雙高風險
    • 🟠 右下角 = 高貸放比 + 低逾放比 → 流動性緊繃
    • 🔵 左下角 = 低貸放比 + 低逾放比 → 資金閒置
    • ⚪ 左上角 = 低貸放比 + 高逾放比 → 逾期風險
💡
使用技巧
— 勾選「在圖表上直接顯示社名」，不用滑鼠移過去也能看到社名
— 圖表下方有純文字列表，列出各狀態的社別名稱

3.3 個社健檢 — 深入看單一社的狀況

選取一間社後，會看到：

  1. 診斷狀態 + 觸發原因（例如「連兩年虧損、貸放比過低」）
  2. 指標對比柱狀圖：該社 vs 區域/全台平均（貸放比、儲蓄率、逾放比、開支比…）
  3. 四張 KPI 數字卡：現有社員、現有股金、逾放比、開支比
  4. 近年指標趨勢圖（年底數據）：開支比、逾放比、貸放比、社員數的多年走勢
⚠️ 趨勢圖注意 如果只有 1 年資料，圖表不會顯示（歷史資料不足）。資料愈多，趨勢愈有參考價值。

3.4 趨勢追蹤 — 看指標是變好還是變壞

可同時比較多間社 + 區域平均，觀察六大指標隨時間的變化：

  • ⚠️ 逾放比趨勢
  • 💰 貸放比趨勢
  • 👥 社員數趨勢
  • 🏦 儲蓄率趨勢
  • 📈 開支比趨勢（只顯示 12 月數據）
  • 🛡️ 提撥率趨勢

預設顯示近 36 個月，您可以自行調整起訖月份。
檢查重點：看線條是往上還是往下，往上不一定壞（要看是什麼指標）。

3.5 報表匯出

可直接在網頁上瀏覽完整報表，或按「📥 匯出 CSV」下載成 Excel 可開的檔案。 報表中高風險社別的「診斷狀態」會自動標色。

4. ⚖️ 財務戰情室

這個頁面專門處理CSV 會計科目明細，適合深入分析財務報表。
左側可以篩選「社別」「年度」「月份」，選好後再看各分頁。

4.1 資產負債表

顯示社在某個月份的資產、負債、權益明細。

  • 左欄：資產部（現金、放款、設備…）
  • 右欄：負債部 + 權益部（社員股金、公積、盈餘…）
  • 自動勾稽：系統會檢查「資產 = 負債 + 權益」，不平衡會顯示紅字警告
📊
進階功能：勾選「對比上月」，可以看每個科目跟一個月前的增減變化。
🔺 綠色 = 增加（資產/權益增加是好，負債增加是壞）
🔻 紅色 = 減少

4.2 綜合損益表

顯示某個年度的收入與支出明細，底部會算出「本期淨利（淨損）」。

  • 收入：利息收入、業務收入…
  • 支出：利息支出、人事費用、業務費用、呆帳提列…
  • 本期淨利 = 收入 - 支出。綠底 = 賺錢、紅底 = 虧損
📊
進階功能：勾選「對比去年」，自動抓取去年同期比較。
如果今年只有 10 個月資料，系統會自動用去年同 10 個月做公平對比。

4.3 年度概覽 — 快速看整年表現

這個分頁會自動產出三種圖表：

  1. KPI 卡片：總資產、總收入、總支出、本期損益、淨值比… 附帶與去年對比
  2. 利潤流向瀑布圖：從總收入開始，扣除各項費用，最後到年度損益，視覺化錢流到哪裡
  3. 歷年收支趨勢：多年度的收入/支出長條圖 + 開支比折線

4.4 深度分析 — 科目排名與異常偵測

兩大功能：

  • 年度科目變動偵測：自動找出跟去年相比金額變動最大的科目，並給出建議（例如「利息支出大增，建議檢視現有借款利率」）
  • 關鍵科目排名：按類別（資產/負債/權益/收入/支出）列出金額 Top 10，可選定對比年度做雙年比較

4.5 財務診斷 — 比紅綠燈更詳細的健康檢查

這個分頁針對單一社別做深入的財務健康診斷，包含：

💰 財務結構

負債比
= 總負債 ÷ 總資產
≤ 80% 🟢 正常 | 80–90% 🟡 偏高 | ≥ 90% 🔴 過高
「借太多錢了，自有資金不足」
淨值比
= 淨值 ÷ 總資產
≥ 20% 🟢 充足 | 10–20% 🟡 偏低 | ≤ 10% 🔴 不足
「自有資本夠不夠厚」

📊 獲利能力

開支比
= 總支出 ÷ 總收入
≤ 95% 🟢 正常 | 95–105% 🟡 偏高 | ≥ 105% 🔴 超標
「賺的錢 cover 得住開銷嗎？」
本期損益
= 收入 – 支出
🟢 正數 = 賺錢 | 🔴 負數 = 虧損
⚠️ 連續兩年虧損 => 緊急警示

📈 歷年趨勢燈號

最下面有一張表格，列出每一年的開支比、加權平均利率、損益，並用紅黃綠燈號標示：

  • 綠底 = 正常
  • 黃底 = 注意
  • 紅底 = 危險

如果歷年中有 2 年以上開支比超過 100%，系統會跳出警示。

5. 🚦 風險分類邏輯與預警應對

系統會自動把每間社分成五種狀態。以下是完整的判斷規則與應對建議：

5.1 五項檢測（觸發任 2 項 → 🚨 特別關懷）

#檢測項目觸發條件代表的意義
① 連兩年虧損 今年和去年開支比都 > 100% 連續兩年入不敷出，體質惡化
② 貸放比過低 貸放比 < 10% 且比去年更低 放款業務萎縮，資金沒有效運用
③ 高逾放且惡化 逾放比 > 50% 且逾期金額比去年增加 放款收不回來且還在惡化
④ 人數連三年衰退 今年人數 < 去年 < 前年 < 大前年 社員持續流失
⑤ 股金連三年衰退 今年股金 < 去年 < 前年 < 大前年 資金持續流出

5.2 五種狀態與應對行動

🚨 特別關懷
觸發 2 項以上檢測。最優先處理。
🔴 建議立即約談該社管理層，擬定改善計畫
⚠️ 流動性緊繃
貸放比 > 90% 且股金負成長
🟠 錢幾乎都放出去了，若股金繼續流失可能週轉不靈。
建議：控制新放款、加強吸收股金
💤 資金閒置
貸放比 < 30% 且逾放比低
🔵 資金沒有有效運用，利息收入偏低。
建議：評估增加放款或投資
✅ 穩健模範
社員/股金成長，貸放比適中，逾放比低
🟢 繼續保持。可作為其他社的學習對象。
📊 一般狀態
未符合上述任何一種狀態
⚪ 各指標平穩或有小波動，持續觀察即可。

5.3 紅綠燈邏輯 — 看懂數字旁邊的箭頭

系統中每個指標旁邊都會有 🔺 或 🔻 加上百分比，顏色的邏輯如下：

🟢 漲好跌壞（紅色上升是好）
  • 社員成長
  • 股金增加
  • 收入增加
  • 淨利增加
  • 淨值比增加

這些指標「越多越好」，所以上升顯示 🔴（好）

🔴 跌好漲壞（綠色下降是好）
  • 開支比
  • 逾放比
  • 負債比
  • 支出

這些指標「越低越好」，所以下降顯示 🔴（好）

⚠️ 容易搞混的重點
— 社員數旁邊出現 🔺 紅字 +5% → 社員增加，是好消息
— 逾放比旁邊出現 🔺 綠字 +2% → 逾放比惡化，是壞消息
簡單記：紅色 = 往好的方向走，綠色 = 往壞的方向走（對多數指標而言）

5.4 預警後的SOP（建議行動流程）

1
看到 🚨 特別關懷
→ 進入「個社健檢」分頁，選該社查看觸發原因
→ 再看「趨勢追蹤」確認是短期波動還是長期惡化
2
到財務戰情室深入分析
→ 切到「⚖️ 財報明細」
→ 看資產負債表（有沒有異常科目）
→ 看綜合損益表（哪一項支出特別高）
→ 看財務診斷（負債比、開支比、歷年趨勢）
3
產出報表與追蹤
→ 在「社務診斷 > 報表匯出」下載 CSV
→ 下次上傳新資料後，確認該社狀態是否有改善

6. 📋 指標速查表

所有指標的計算方式與意義，一次對照：

指標計算方式正常範圍太高/太低代表
貸放比 放款總額 ÷ 股金總額 40% – 80% >90% 流動性緊繃，<30% 資金閒置
逾放比 逾期放款 ÷ 總放款 < 2% >10% 高風險，>50% 嚴重
開支比 總支出 ÷ 總收入 < 95% >100% 虧損，>105% 嚴重
負債比 總負債 ÷ 總資產 < 80% >90% 自有資金嚴重不足
淨值比 淨值 ÷ 總資產 > 20% <10% 資本嚴重不足
儲蓄率 儲蓄存款 ÷ 股金 > 60% 過低代表吸收儲蓄能力差
提撥率 備抵呆帳 ÷ 放款 > 1% 過低代表風險準備不足

股金成長率 (本期 – 前期) ÷ 前期 > 0% 負數代表資金流出

7. ❓ 常見問題

Q: 為什麼我看不到「⚖️ 財報明細」？

A: 需要上傳 CSV 檔案（或分享連結中有包含 csv=xxx）。只有上傳 Excel 的話，只會出現「社務診斷」。

Q: 箭頭顏色好亂，怎麼區分好壞？

A: 簡單記：🔴 紅色上升 = 好事（社員、股金、收入、淨利、淨值比）；🟢 綠色上升 = 壞事（開支比、逾放比、負債比、支出）。
所以看到紅箭頭先放心，綠箭頭要留意。

Q: 上傳後資料沒更新？

A: 系統有暫存機制，重新上傳同一個檔案可能還會看到舊資料。可以重整頁面（F5）再上傳一次。

Q: 風險矩陣的泡泡好擠，社名都疊在一起？

A: 勾選「在圖表上直接顯示社名」會顯示名稱；如果不勾，滑鼠移到泡泡上也會跳出社名。

Q: 趨勢圖只有一個社的線，看不到平均值？

A: 勾選「顯示區域平均/全台平均」，就會多一條虛線可以對照。

Q: 如何分享給其他人看？

A: 管理員上傳資料後，按側邊欄「生成分享連結」，複製網址傳給對方。對方打開連結、輸入自己的密碼就能看到資料。

Q: 手機上字太小 / 按鈕不好按？

A: 本系統已針對手機優化：字體 18px、按鈕高度 52px、側邊欄採用抽屜式設計。如果還是太小，建議橫向使用或使用平板/電腦。
